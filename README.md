#Search engine simulation

Final assignment in course Operating Systems 1.

**[Link to assignment](http://www.elad-horev.org/Teaching/OS1/2016b/Assignment/Assignment.pdf)**

## Abstract
 In this assignment you will be simulating the functionality of
a search engine. You will implement both the client and server side. The
assignment puts an emphasis on multithreaded programming and the synchronisation
of these which is the main theme of our course. The assignment
involves the use of sockets
